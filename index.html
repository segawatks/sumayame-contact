選択したアプリがブロックされない / 選んでいないアプリがブロックされるとき Selected apps aren't blocked / wrong apps are blocked

Apple のスクリーンタイム側の不具合により、本アプリに限らず類似アプリ（Forest 等）でも発生することがあります。以下をお試しください。

  • ブロック対象のアプリを一度解除し、選び直す（最も効果が高いです）
  • iPhone 以外の端末（iPad / Mac 等）のスクリーンタイムをオフにする
  • iOS を最新バージョンにアップデートする
  • iPhone を再起動する

それでも解消しない場合は、上記の「アプリ内の問い合わせボタン」からご連絡ください。

This is a known issue with Apple's Screen Time API that affects many similar apps (including Forest). Please try the following:

  • Re-select the blocked apps — remove them once and pick them again (most effective)
  • Turn Screen Time off on other devices (iPad / Mac)
  • Update iOS to the latest version
  • Restart your iPhone

If it still doesn't resolve, please contact me through the in-app contact button above.
